# Lab 03 - Self-Service Password Reset

## Objective

Configure Self-Service Password Reset (SSPR) in Microsoft Entra ID to allow users to securely reset their passwords.

## Environment

- Microsoft 365 E5
- Microsoft Entra ID

## Configuration

- Enabled Self-Service Password Reset
- Configured authentication methods
- Required user registration
- Configured notification settings

## Validation

Verified that SSPR was enabled and authentication methods were correctly configured.

## Skills Demonstrated

- Microsoft Entra ID
- Self-Service Password Reset
- Authentication Methods
- Identity Security
- User Self-Service
