# Lab 05 - Administrative Units & RBAC

## Objective

Configure an Administrative Unit in Microsoft Entra ID and demonstrate delegated administration using Role-Based Access Control (RBAC).

## Environment

- Microsoft 365 E5
- Microsoft Entra ID

## Configuration

- Created an Administrative Unit
- Added users to the Administrative Unit
- Assigned a scoped administrative role
- Demonstrated delegated administration principles

## Validation

Verified the Administrative Unit, member assignments, and scoped role assignment.

## Skills Demonstrated

- Microsoft Entra ID
- Administrative Units
- Role-Based Access Control (RBAC)
- Delegated Administration
- Least Privilege
