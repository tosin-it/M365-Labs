# Lab 04 - Authentication Methods

## Objective

Configure modern authentication methods in Microsoft Entra ID, including Microsoft Authenticator, Temporary Access Pass, and FIDO2 Security Keys.

## Environment

- Microsoft 365 E5
- Microsoft Entra ID

## Configuration

- Enabled Microsoft Authenticator
- Enabled Temporary Access Pass
- Enabled FIDO2 Security Keys (if supported)
- Generated a Temporary Access Pass for a test user

## Validation

Verified authentication method policies and confirmed a Temporary Access Pass was successfully issued.

## Skills Demonstrated

- Microsoft Entra ID
- Authentication Methods
- Passwordless Authentication
- Temporary Access Pass
- Identity Security
