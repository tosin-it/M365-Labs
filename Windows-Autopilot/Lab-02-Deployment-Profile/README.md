# Lab 02 - Windows Autopilot Deployment Profile

## Objective

Configure and validate a Windows Autopilot deployment profile for Microsoft Entra joined devices.

## Environment

- Microsoft 365 E5
- Microsoft Intune
- Windows 11 VM

## Configuration

- Configured a user-driven deployment profile
- Assigned the profile to Autopilot devices
- Customized the Out-of-Box Experience (OOBE)
- Configured Microsoft Entra Join
- Verified deployment profile assignment

## Validation

Verified that the deployment profile was successfully assigned to the Autopilot device and that the device reported an enrolled state.

## Skills Demonstrated

- Windows Autopilot
- Deployment Profiles
- OOBE Configuration
- Microsoft Entra Join
- Microsoft Intune
