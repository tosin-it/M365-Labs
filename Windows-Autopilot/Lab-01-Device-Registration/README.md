# Lab 01 - Windows Autopilot Device Registration

## Objective

Register a Windows device with Windows Autopilot and validate successful import.

## Environment

- Microsoft 365 E5
- Microsoft Intune
- Windows 11 VM

## Configuration

- Collected hardware hash
- Imported device into Windows Autopilot
- Verified successful registration

## Validation

Confirmed the device appeared in the Windows Autopilot devices list with the expected hardware details.

## Skills Demonstrated

- Windows Autopilot
- Device Registration
- Endpoint Provisioning
- Microsoft Intune
