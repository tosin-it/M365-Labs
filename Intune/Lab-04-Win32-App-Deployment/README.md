# Lab 04 - Win32 Application Deployment

## Objective

Deploy a Win32 application using Microsoft Intune to managed Windows 11 devices.

## Environment

- Microsoft 365 E5
- Microsoft Intune
- Microsoft Entra ID
- Windows 11 VM

## Application

- 7-Zip

## Configuration

- Packaged application using Microsoft Win32 Content Prep Tool
- Uploaded .intunewin package to Microsoft Intune
- Configured silent installation and uninstallation commands
- Created manual file-based detection rule
- Assigned application as Required to a test device group

## Validation

- Application successfully deployed to the enrolled Windows 11 VM
- Installation status reported as Installed in Microsoft Intune

## Skills Demonstrated

- Win32 application packaging
- Microsoft Intune application deployment
- Detection rule configuration
- Endpoint software management
- Deployment monitoring
