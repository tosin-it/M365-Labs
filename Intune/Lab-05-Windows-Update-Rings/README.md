# Lab 05 - Windows Update Rings

## Objective

Configure and deploy a Windows Update Ring using Microsoft Intune to manage Windows Update for Business settings on enrolled Windows 11 devices.

## Environment

- Microsoft 365 E5
- Microsoft Intune
- Microsoft Entra ID
- Windows 11 VM

## Configuration

- Enabled Microsoft product updates
- Enabled Windows driver updates
- Configured 7-day quality update deferral
- Configured 14-day feature update deferral
- Configured automatic installation and restart during maintenance windows
- Defined active hours to minimize user disruption

## Assignment

Assigned the Update Ring to a Windows 11 test device group.

## Validation

- Policy successfully assigned
- Device synchronized with Microsoft Intune
- Update Ring status reported successfully

## Skills Demonstrated

- Windows Update for Business
- Intune Update Rings
- Patch Management
- Endpoint Administration
- Windows Update Policy Management
