# Lab 02 - Windows Configuration Profiles

## Objective

Deploy a Windows configuration profile using Microsoft Intune to enforce baseline security settings on managed Windows 11 devices.

## Environment

- Microsoft 365 E5
- Microsoft Intune
- Microsoft Entra ID
- Windows 11 VM

## Configuration

- Password policy
- Microsoft Defender enabled
- Windows Firewall enabled
- Windows Defender SmartScreen enabled via Administrative Templates (OS level)

## Deployment

The profile was assigned through Microsoft Intune and successfully applied to the enrolled Windows 11 virtual machine.

## Validation

- Policy deployed successfully
- Device synchronized
- Configuration profile reported a successful status

## Skills Demonstrated

- Microsoft Intune
- Configuration Profiles
- Windows Security Policies
- Endpoint Administration
- Policy Deployment
