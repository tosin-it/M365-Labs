# Lab 03 - Device Compliance Policies

## Objective
Configure and enforce device compliance policies using Microsoft Intune to ensure Windows devices meet organizational security requirements.

## Environment
- Microsoft 365 E5
- Microsoft Intune
- Microsoft Entra ID
- Windows 11 VM

## Configuration

- Required BitLocker encryption
- Secure Boot enforcement
- Device health attestation
- Microsoft Defender Antivirus required
- Windows Firewall required
- Password policy enforcement required

## Assignment
Policy assigned to test device group for validation.

## Validation
- Device evaluated against compliance policy
- Compliance status reported in Intune
- Device successfully evaluated after sync

## Skills Demonstrated
- Device compliance configuration
- Security posture enforcement
- Endpoint governance
- Intune policy management
