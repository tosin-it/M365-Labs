# Lab 01 - Windows Device Enrollment into Microsoft Intune

## Objective

Enroll a Windows 11 virtual machine into Microsoft Intune and verify successful device registration and management.

## Environment

- Microsoft 365 E5 
- Microsoft Intune
- Microsoft Entra ID
- Windows 11 Virtual Machine
- Hyper-V

## Tasks Completed

- Connected Windows 11 VM to Microsoft Entra ID
- Successfully enrolled device into Microsoft Intune
- Verified device synchronization
- Confirmed compliance status
- Verified device registration in Microsoft Entra ID

## Validation

The device successfully appeared in:

- Microsoft Intune Admin Center
- Microsoft Entra Admin Center

The device is now available for policy deployment, application deployment, compliance evaluation, and remote management.

## Skills Demonstrated

- Microsoft Intune Administration
- Windows Device Enrollment
- Microsoft Entra ID Integration
- Endpoint Management
- Device Lifecycle Management
