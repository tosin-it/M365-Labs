## Project Overview

This repository documents a complete Microsoft 365 enterprise lab environment built using a Microsoft 365 E5 tenant. The objective was to simulate real-world endpoint administration, cloud identity management, automation, and security operations commonly performed by Microsoft 365 Administrators and Endpoint Engineers.

The portfolio focuses on practical implementation rather than theoretical exercises. Each lab includes documented configurations, PowerShell automation, generated reports, screenshots, and implementation notes.
