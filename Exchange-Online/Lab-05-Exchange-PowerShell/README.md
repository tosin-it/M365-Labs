# Lab 05 - Exchange Online PowerShell Administration

## Objective

Use Exchange Online PowerShell to automate mailbox administration and reporting.

## Environment

- Microsoft 365 E5
- Exchange Online
- PowerShell 5
- ExchangeOnlineManagement Module

## Configuration

- Connected to Exchange Online
- Generated mailbox inventory
- Generated shared mailbox permission report
- Exported results to CSV

## Reports

- MailboxInventory.csv
- SharedMailboxPermissions.csv

## Skills Demonstrated

- Exchange Online PowerShell
- Reporting Automation
- Shared Mailbox Administration
- PowerShell Scripting
- Microsoft 365 Administration
