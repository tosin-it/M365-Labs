# Lab 01 - Shared Mailboxes

## Objective

Create and configure a Shared Mailbox in Exchange Online and delegate mailbox permissions.

## Environment

- Microsoft 365 E5
- Exchange Online
- Microsoft Entra ID

## Configuration

- Created a Shared Mailbox
- Assigned Full Access permissions
- Assigned Send As permissions
- Validated mailbox configuration

## Validation

Verified mailbox creation and delegated permissions in the Exchange Admin Center.

## Skills Demonstrated

- Exchange Online
- Shared Mailboxes
- Mailbox Delegation
- Microsoft 365 Administration
