# Lab 04 - Message Trace & Mail Flow Troubleshooting

## Objective

Use Exchange Online Message Trace to investigate email delivery and validate mail flow rules.

## Environment

- Microsoft 365 E5
- Exchange Online

## Configuration

- Generated internal and external test messages
- Traced message delivery
- Verified transport rule execution
- Exported message trace results

## Validation

| Scenario | Result |
|----------|--------|
| Internal Email | Delivered |
| External Email | Delivered with warning |
| Blocked Attachment | Rejected |

## Skills Demonstrated

- Exchange Online

- Email Delivery Analysis
- Exchange Administration
