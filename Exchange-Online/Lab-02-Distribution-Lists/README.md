# Lab 02 - Distribution Lists & Microsoft 365 Groups

## Objective

Create and manage Exchange Online Distribution Lists and Microsoft 365 Groups while comparing their functionality.

## Environment

- Microsoft 365 E5
- Exchange Online
- Microsoft Entra ID

## Configuration

- Created a Distribution List
- Added members
- Created a Microsoft 365 Group
- Configured group membership
- Compared collaboration capabilities

## Comparison

| Feature | Distribution List | Microsoft 365 Group |
|---------|-------------------|---------------------|
| Email Distribution | Yes | Yes |
| Shared Mailbox | No | Yes |
| Shared Calendar | No | Yes |
| SharePoint Site | No | Yes |
| Teams Integration | No | Yes |

## Validation

Verified both group types, membership, and associated resources.

## Skills Demonstrated

- Exchange Online
- Distribution Lists
- Microsoft 365 Groups
- Recipient Management
- Microsoft 365 Collaboration
