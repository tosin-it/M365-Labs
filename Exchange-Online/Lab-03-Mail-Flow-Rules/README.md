# Lab 03 - Mail Flow Rules

## Objective

Create Exchange Online mail flow rules to identify external email and block high-risk attachments.

## Environment

- Microsoft 365 E5
- Exchange Online

## Configuration

- Created an External Email Warning rule
- Prepended "[EXTERNAL]" to messages from outside the organization
- Created a rule to block executable attachments
- Configured sender notification for rejected messages

## Validation

- Verified external email subject modification
- Verified attachment blocking behavior
- Confirmed rule processing order

## Skills Demonstrated

- Exchange Online
- Mail Flow Rules
- Transport Rules
- Email Security
- Microsoft 365 Administration
