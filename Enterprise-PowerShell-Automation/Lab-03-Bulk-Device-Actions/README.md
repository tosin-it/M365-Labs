# Lab 03 - Bulk Intune Device Actions

## Objective

Use Microsoft Graph PowerShell to automate bulk device synchronization for Microsoft Intune managed devices.

## Environment

- Microsoft 365 E5
- Microsoft Intune
- Microsoft Graph PowerShell SDK

## Configuration

- Connected to Microsoft Graph
- Retrieved managed devices
- Executed bulk device synchronization
- Exported execution results to CSV

## Reports

- BulkDeviceSyncResults.csv

## Skills Demonstrated

- Microsoft Graph PowerShell
- Microsoft Intune
- Device Management
- Bulk Automation
- Endpoint Administration

## Future Enhancements

The same approach can be extended to automate:

- Device Restart
- Remote Lock
- Device Rename
- Autopilot Device Actions
