# Lab 01 - Microsoft 365 Environment Health Dashboard

## Objective

Develop a PowerShell solution that collects key Microsoft 365 tenant information and exports inventory reports.

## Environment

- Microsoft 365 E5
- Microsoft Graph PowerShell SDK

## Configuration

- Connected to Microsoft Graph
- Collected tenant information
- Retrieved user, device, and group counts
- Exported license inventory
- Generated CSV reports

## Reports

- TenantSummary.csv
- LicenseInventory.csv

## Skills Demonstrated

- Microsoft Graph PowerShell
- Microsoft 365 Reporting
- PowerShell Automation
- Tenant Administration
