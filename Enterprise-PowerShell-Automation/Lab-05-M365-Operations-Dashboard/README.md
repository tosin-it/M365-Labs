# Lab 05 - Microsoft 365 Operations Dashboard

## Objective

Create a unified Microsoft 365 operations dashboard by collecting tenant information from Microsoft Graph and Exchange Online and presenting it in an HTML report.

## Environment

- Microsoft 365 E5
- Microsoft Graph PowerShell SDK
- Exchange Online PowerShell

## Configuration

- Connected to Microsoft Graph
- Connected to Exchange Online
- Collected tenant, user, group, device, compliance, mailbox, and license information
- Generated an HTML operations dashboard

## Report

- M365OperationsDashboard.html

## Skills Demonstrated

- Microsoft Graph PowerShell
- Exchange Online PowerShell
- Microsoft Intune
- Microsoft Entra ID
- HTML Reporting
- Cross-Service Automation
- Enterprise Reporting
