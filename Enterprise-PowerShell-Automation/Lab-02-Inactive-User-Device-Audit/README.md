# Lab 02 - Inactive User & Device Audit

## Objective

Create a PowerShell solution to inventory Microsoft Entra users and Microsoft Intune managed devices, producing CSV reports for administrative review.

## Environment

- Microsoft 365 E5
- Microsoft Graph PowerShell SDK

## Configuration

- Connected to Microsoft Graph
- Retrieved Microsoft Entra users
- Retrieved Intune managed devices
- Generated compliance summary
- Exported reports to CSV

## Reports

- UserInventory.csv
- ManagedDevices.csv
- ComplianceSummary.csv

## Skills Demonstrated

- Microsoft Graph PowerShell
- Microsoft Intune
- Microsoft Entra ID
- Inventory Reporting
- PowerShell Automation
