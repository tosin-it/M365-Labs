# Lab 04 - Compliance & Security Dashboard

## Objective

Develop a PowerShell solution that generates Microsoft Intune compliance reports in CSV and HTML formats.

## Environment

- Microsoft 365 E5
- Microsoft Intune
- Microsoft Graph PowerShell SDK

## Configuration

- Connected to Microsoft Graph
- Retrieved managed device compliance information
- Generated CSV report
- Generated HTML executive dashboard

## Reports

- ComplianceReport.csv
- ComplianceDashboard.html

## Skills Demonstrated

- Microsoft Graph PowerShell
- HTML Reporting
- Device Compliance
- PowerShell Automation
- Executive Reporting
