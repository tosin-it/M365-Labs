# Lab 01 - Managed Device Inventory (Microsoft Graph)

## Objective

Use Microsoft Graph PowerShell to retrieve managed device information from Microsoft Intune and export the results to a CSV report.

## Environment

- Microsoft 365 E5
- Microsoft Graph PowerShell SDK
- Microsoft Intune
- Windows 11

## Configuration

- Connected to Microsoft Graph
- Queried managed devices
- Exported inventory to CSV
- Validated device information against Microsoft Intune

## Skills Demonstrated

- Microsoft Graph PowerShell
- Intune automation
- Device inventory reporting
- CSV export
- Endpoint administration
