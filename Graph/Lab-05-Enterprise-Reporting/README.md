# Lab 05 - Enterprise Reporting

## Objective

Generate Microsoft 365 reports using Microsoft Graph PowerShell for users, security groups, managed devices, and license assignments.

## Environment

- Microsoft 365 E5
- Microsoft Graph PowerShell SDK
- Microsoft Intune
- Microsoft Entra ID
- Windows 11

## Reports Generated

- Users
- Security Groups
- Managed Devices
- User License Assignments

## Configuration

- Connected to Microsoft Graph
- Queried Microsoft Entra ID users
- Queried Microsoft Entra ID groups
- Queried Microsoft Intune managed devices
- Exported reports to CSV

## Validation

Verified successful report generation and reviewed exported CSV files.

## Skills Demonstrated

- Microsoft Graph PowerShell
- Enterprise reporting
- CSV reporting
- Microsoft Intune
- Microsoft Entra ID
- PowerShell automation
