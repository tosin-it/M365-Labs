# Lab 02 - Bulk User Provisioning

## Objective

Provision multiple Microsoft Entra ID users using Microsoft Graph PowerShell and a CSV file.

## Environment

- Microsoft 365 E5
- Microsoft Graph PowerShell SDK
- Microsoft Entra ID
- Windows 11

## Configuration

- Imported user data from CSV
- Connected to Microsoft Graph
- Created Microsoft Entra ID users
- Configured temporary passwords
- Enabled force password change at first sign-in
- Implemented duplicate user detection
- Added error handling and execution summary

## Validation

- Successfully created users in Microsoft Entra ID
- Verified user properties
- Confirmed duplicate users were skipped

## Skills Demonstrated

- Microsoft Graph PowerShell
- Identity automation
- Bulk user provisioning
- CSV processing
- Error handling
